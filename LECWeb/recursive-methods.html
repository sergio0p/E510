Recursive Methods

How can we compute these infinite sums without doing an infinite number of summations or using Mathematica?

$$\sum_{t=0}^{\infty}\delta^{t}=\text{?}\quad \sum_{t=T}^{\infty}\delta^{t}=\text{?}$$

By using recursive methods:

  • Let $A=1+\delta+\delta^{2}+\delta^{3}+\cdots$
  • Factor out $\delta$: $A=1+\delta\cdot(1+\delta+\delta^{2}+\cdots)$
  • Notice the expression in parentheses equals $A$: $A=1+\delta\cdot A$
  • This is a recursive equation: $A$ is defined in terms of $A$
  • We can easily solve: $A-\delta A=1$
  • Therefore: $(1-\delta)A=1$
  • Solution: $A=\dfrac{1}{1-\delta}$

Recursive Methods

An Example

Chuck Noland is castaway on a desert island forever and has $K$ units of perfectly divisible rations. He has to choose how many rations to consume in each period to maximize his discounted utility so, he solves the problem:

$$\max\limits_{\substack{\phantom{X}\\[3pt](c_t)_{t=0}^{\infty}\\[4pt] \text{subject to} \\[4pt] \sum_{t=0}^{\infty}c_t= K }} \sum_{t=0}^{\infty}\delta^{t}u(c_t)$$

We could apply recipe #1 here but then we would have to solve an infinite number of equations for an infinite number of unknowns!

Recursive Methods Example

cont.

Let $c_t^\ast(K)$ be the optimal consumption at time $t$ given that Chuck has $K$ units of rations at the start. If we plug back the optimal consumption path into the discounted utility function we get the value of the maximization problem. Remember that this is (by definition) the value function:

$$\begin{aligned} V(K)&=\sum_{t=0}^{\infty}\delta^{t}u(c_t^\ast(K))\\ &=\max\limits_{\substack{\phantom{X}\\[3pt](c_t)_{t=0}^{\infty}\\[4pt] \text{subject to} \\[4pt] \sum_{t=0}^{\infty}c_t= K }} \sum_{t=0}^{\infty}\delta^{t}u(c_t) \end{aligned}$$

Recursive Methods Example

The Optimality Principle

Suppose that, in period zero, Chuck consumes $c_0$ which might be a non-optimal amount. Then, he has $K-c_0$ units of rations left for the future periods.

Also, suppose that from period one onwards, Chuck only makes optimal choices. Notice the optimal choices would have changed due to the period zero choice. His discounted utility can be expressed as:

$$u(c_0)+\delta\cdot V(K-c_0)$$

Moreover,

$$V(K)\ge u(c_0)+\delta\cdot V(K-c_0)$$

because $V(K)$ is the maximum discounted utility that Chuck can get with $K$ units of rations. This is called the optimality principle.

Recursive Methods Example

The Recursive Equation

By the optimality principle we have that:

$$V(K)\ge u(c_0)+\delta\cdot V(K-c_0)$$

and since this is true for any $c_0$ such that $0\le c_0\le K$ we have that:

$$V(K)=\max\limits_{0\le c\le K} u(c)+\delta\cdot V(K-c)$$

This is the recursive equation for the value function $V(K)$.

The Optimal Consumption Policy

Example cont.

We refer to the function $c^\ast(K)$ that gives the optimal consumption in the current period as a function of the current level of rations $K$ as the policy function. Notice that the policy function does not depend on the calendar time $t$ because the problem is time-invariant. Chuck does not care which period it is, he only cares about how many rations he has left.

Using the policy function we can obtain the value function as:

$$\begin{aligned} V(K)=&\;u(c^\ast(K))+\delta\cdot u(c^\ast(K-c^\ast(K)))+\\ &+\delta^{2}\cdot u(c^\ast(K-c^\ast(K)-c^\ast(K-c^\ast(K))))+\cdots \end{aligned}$$

Conversely, if we know the value function we can obtain the policy function as the solution of the following first order condition:

$$u^\prime(c)-\delta\cdot V^\prime(K-c)=0.$$

Solving the Recursive Equation

Under certain conditions, any solution of the recursive equation is the value function of the original problem. To solve the recursive equation we can use value function iteration:

  • Start with an initial guess for the value function, $V_0(K)$.
  • Solve the first order condition $u^\prime(c)-\delta\cdot V_0^\prime(K-c)=0$ to obtain the corresponding policy function $c_0^\ast(K)$.
  • Plug $c_0^\ast(K)$ into $u(c)+\delta\cdot V_0(K-c)$ to obtain a new value function $V_1(K)= u(c_0^\ast(K))+\delta\cdot V_0(K-c_0^\ast(K))$.
  • Repeat the process until convergence.
  • Under certain conditions, this process converges to the value function of the original problem.
