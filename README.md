# ECON 510: Advanced Microeconomic Theory

This repository contains materials for Lectures on Dynamic Microeconomics

## Structure

- `syllabus/`: Course syllabus
- `modules/`: Lecture notes, slides, Mathematica files
- `problem-sets/`: Homework and solutions
- `exams/`: Midterm and final materials
- `scripts/`: Helper scripts (LaTeX macros, grading tools, etc.)
