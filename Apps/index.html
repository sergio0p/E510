Interactive Apps

ECON 510 - Advanced Microeconomic Theory - Spring 2026

Instructor: Sérgio O. Parreiras

These interactive web applications help you understand key concepts in advanced microeconomic theory through hands-on practice and visualization.

Backward Induction Game Tree

Topics: Game Theory, Backward Induction, Sequential Games, Subgame Perfect Equilibrium

Description: Visualize and solve extensive-form games using backward induction. This interactive app helps you understand:

  • Game tree structure and sequential decision-making
  • How to find subgame perfect Nash equilibria
  • Best response dynamics in sequential games
  • The power of commitment and credibility in strategic situations

Features: Animated tree generation, interactive node selection, step-by-step solution visualization

Launch App →

Discounting & Dynamic Consistency

Topics: Intertemporal Choice, Exponential Discounting, Hyperbolic Discounting, Time Inconsistency

Description: Practice calculating present utility and test for dynamic consistency of preferences. This app helps you understand:

  • How to calculate present utility with exponential and hyperbolic discounting
  • The difference between time-consistent and time-inconsistent preferences
  • Why hyperbolic discounting leads to preference reversals
  • Applications to commitment problems and self-control

Features: Random problem generation, exponential vs hyperbolic discounting, dynamic consistency checks, detailed step-by-step solutions

Launch App →

Recursive Utility Calculator

Topics: Recursive Preferences, Aggregator Functions, Intertemporal Substitution, Risk Aversion

Description: Calculate and compare recursive utility functions with different parameter specifications. This app helps you understand:

  • How recursive utility separates risk aversion from intertemporal substitution
  • The role of the aggregator function in preference formation
  • Epstein-Zin preferences and their applications
  • Comparative statics with respect to preference parameters

Features: Interactive parameter controls, real-time calculations, comparison across specifications, detailed mathematical explanations

Launch App →

Arbitrage & Asset Pricing

Topics: Financial Markets, Arbitrage, Complete Markets, State Prices, Arrow-Debreu Securities

Description: Identify arbitrage opportunities and calculate state prices in complete markets. This app helps you understand:

  • What constitutes an arbitrage opportunity
  • How to construct arbitrage portfolios
  • The relationship between asset prices and state prices
  • Market completeness and spanning

Features: Random market scenarios, arbitrage detection, state price calculation, portfolio construction guidance
