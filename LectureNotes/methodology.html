Microeconomic Theory Methodology

  • Individual agents (consumers, firms, government, etc...) are the basis of the analysis.
  • Agents act accordingly to their preferences (i.e. consumers maximize utility, firms maximize profits, etc...)
  • Equilibrium (agents chose optimal actions given the actions of the others).
  • Exogenous variables (variables that describe the environment in which agents operate, they are given/constants for practical purposes).
  • Endogenous variables (variables that are chosen/set according to the agents' choices).
  • Comparative statics: how the equilibrium value of the endogenous variables changes when the exogenous variables change.
  • If two models deliver the same predictions, stick with the simplest one (Occam's razor).
  • Descriptive model: tells how agents behave.
  • Normative model: tells how agents should behave.

Microeconomic Theory Methodology

  • All economic models are wrong!
  • ``Realism'' is not a good metric to evaluate the quality of a model.
  • Forecast ability is a better metric but there are others ...
  • If two models deliver the same predictions, stick with the simplest one (Occam's razor).
  • Building a model is like creating an impressionist painting. The artist needs to decide which details to include in the model and which details to exclude.
  • Details may generate dramatically different policy recommendations. Example, if externality are absent, government intervention in a market leads to inefficiencies. But if externalities are present, government intervention may improve overall welfare...
  • If a detail is important or not, often can only be answered with data and not by theoretical means.

Microeconomic Theory Methodology

Building and analyzing models

  1. Set-up the environment: agents, preferences, and constraints (exogenous variables).
  2. Obtain the agents optimal actions given the environment and choices of other agents (optimal choice or best-response, equilibrium condition).
  3. Require choices of agents to be consistent (equilibrium condition).
  4. Comparative statics (how the equilibrium changes with the changes in the endogenous variables).
  5. Use data to test the model.
